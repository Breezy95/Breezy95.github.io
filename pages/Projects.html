Projects

Here are some unique projects that I have been working on accompanied by a small description. I picked these because they all challenged me in different ways and forced me to think differently about how I solved problems

Network Simulator

This was a project that relied primarily on more creative usage of lists, stacks and queues. I used several methods that would help emulate arrival probability of packets, information buffer for the packets and also calculated some basic statistics such as arrival time per packet, maximum and minimum packet size. click here or image for github

Scribd Crawler

This project was made to see if I could come up with some way to bypass the protection that scribd places on its books. This version of it is rudimentary and uses screenshots to capture the information. Later versions will attempt to parse text to recreate a more faithful image of the page

Packages/Libraries used:
  • Python 3.0
  • BeautifulSoup4
  • Selenium (using Chrome) for python
  • py2pdf python library
oNline Web Fonts
